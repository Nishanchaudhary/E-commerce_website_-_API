# E-commerce Website API

This repository contains the backend API for an e-commerce platform, providing essential functionalities for user authentication, product management, order processing, and more.

## Features

- **User Authentication**: Secure user registration and login.
- **Product Management**: CRUD operations for products.
- **Order Processing**: Manage customer orders and order statuses.
- **Admin Panel**: Administrative access for managing users, products, and orders.

## Technologies Used

- **Backend Framework**: Django REST framework
- **Database**: SQLITE
- **Authentication**: JSON Web Tokens (JWT)
